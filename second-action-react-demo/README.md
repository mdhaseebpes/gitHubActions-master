# second-action-react-demo

A small **React + Vite** sample project designed for learning **GitHub Actions** CI.

## Scripts
- `npm install`
- `npm run dev`
- `npm test`
- `npm run build`

## GitHub Actions
- `.github/workflows/ci.yml` runs tests + build on PRs and pushes to `main`.
- `.github/workflows/deploy-pages.yml` (optional) deploys to GitHub Pages on push to `main`.
